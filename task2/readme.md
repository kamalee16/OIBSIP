# Advanced BMI Calculator

## Objective
The objective of this project is to develop an Advanced BMI Calculator using Python that calculates Body Mass Index (BMI), categorizes the result, stores user data, and displays BMI history graphically.

## Steps Performed
1. Designed a GUI using Tkinter.
2. Collected user input for name, weight, and height.
3. Validated inputs and handled errors.
4. Converted height from centimeters to meters when required.
5. Calculated BMI using the standard formula.
6. Categorized BMI into Underweight, Normal, Overweight, and Obese.
7. Stored BMI records in a CSV file with date.
8. Displayed BMI history using a line graph.

## Tools Used
- Python  
- Tkinter  
- CSV Module  
- Matplotlib  
- Datetime  
- OS Module  

## Outcome
- Calculates BMI accurately and displays the health category.
- Stores BMI data for multiple users.
- Shows BMI trend over time using a graph.
- Provides a simple and user-friendly interface.

## How to Run
1. Install Python.
2. Install matplotlib:
   pip install matplotlib
3. Run the program:
   python bmi_calculator.py
