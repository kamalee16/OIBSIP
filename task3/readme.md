## Objective
The objective of this project is to create a user-friendly **Advanced Password Generator** using Python that allows users to generate strong and secure passwords. The application provides customization options such as password length, character types, exclusion of specific characters, password strength checking, and clipboard copying.

---

## Steps Performed
1. Designed a graphical user interface (GUI) using **Tkinter**.
2. Implemented password generation logic using random character selection.
3. Added options to include:
   - Lowercase letters
   - Uppercase letters
   - Numbers
   - Symbols
4. Implemented functionality to exclude specific characters from the password.
5. Added validation for password length and character selection.
6. Integrated a password strength checker (Weak / Moderate / Strong).
7. Added clipboard functionality to copy the generated password easily.
8. Displayed appropriate alert messages for errors and actions.

---

## Tools Used
- Python
- Tkinter (for GUI development)
- Random module (for password generation)
- String module (for character sets)

---

## Outcome
The project successfully generates secure and customizable passwords through an easy-to-use graphical interface. Users can assess password strength, exclude unwanted characters, and copy passwords to the clipboard instantly. This tool enhances password security and usability for everyday applications.
